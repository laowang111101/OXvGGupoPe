# 房屋租赁管理系统 java440

## 项目概述

房屋租赁管理系统是基于B/S架构和MVC设计模式，采用前后端分离的开发模式构建。系统旨在为管理员和租客提供便捷的房屋租赁管理服务。

## 技术栈

### 系统环境
- Windows/Mac

### 开发环境
- IntelliJ IDEA
- JDK 1.8
- Maven

### 前端技术栈
- Vue
- Axios
- Element-ui
- Echarts

### 后端技术栈
- Java
- SpringBoot
- Mybatis
- MySQL 5.7

## 功能模块

1. **用户登录**
   - 管理员和租客可登录进入对应的后台管理界面。

2. **房屋信息管理**
   - 管理房屋租客信息。

3. **租金管理**
   - 添加租金信息，包括租客姓名、地址、金额、缴纳日期等。
   - 管理租金缴纳状态。

4. **故障管理**
   - 查看并处理租客上报的房屋故障信息。

5. **新闻公告管理**
   - 发布新闻公告，告知租客相关信息。

6. **用户管理**
   - 管理员可以添加租客，设置用户角色。

7. **租客登录**
   - 租客可以查看房屋信息，进行故障申报等。

## 系统角色
- 管理员：负责房屋租赁的整体管理。
- 租客：查看房屋信息，管理个人租赁事务。

## 运行环境
- 前端：现代浏览器。
- 后端：Java运行环境，MySQL数据库。

## 部署步骤
1. 环境准备：安装Java JDK 1.8、Maven、MySQL 5.7。
2. 克隆或导入源码到IDEA。

4. 配置项目相关参数。
5. 分别编译和打包前后端代码。
6. 部署至服务器。
